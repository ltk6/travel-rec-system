# N6: Activity Ranking

Ranks activities generated by N5 using semantic similarity and constraint matching.

## API
```python
rank_activities(data: dict) -> dict
```

### Input
- `user_vectors`: Multimodal embeddings (tags, context, emotions)
- `activities`: List of candidates from N5
- `constraints`: Budget & duration limits
- `context`: Real-time factors (time of day, weather)
- `top_k`: Number of results to return (default: 5)

### Output
- `activities`: Top-K ranked activities with:
  - `score`: Combined relevance (0.0 - 1.0)
  - `reason`: Natural language explanation for the rank.

## Logic
1. **Hard Constraints**: Filters out activities exceeding the user's available duration.
2. **Semantic Score (50%)**: Cosine similarity between user intent vectors and activity descriptors.
3. **Constraint Score (25%)**: Penalty for high cost or poor time-fit.
4. **Context Score (25%)**: Boosts activities matching current weather and time of day.
